# AI Assistant Instructions (Gemini / Antigravity)

Whenever you make any changes to this codebase, you MUST execute the following verification steps:

1. **Format**: Format the code by running:
   ```bash
   cargo fmt
   ```
2. **Test**: Run the test suite using the Makefile target:
   ```bash
   make test
   ```
3. **Build**: Rebuild the WebAssembly package using the Makefile target:
   ```bash
   make build
   ```
